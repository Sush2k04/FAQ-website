Width/height includes content, padding, and border The box maintains the exact specified dimensions Formula: content width = width - padding - border
Width/height only includes the content area Padding and border are added to the specified dimensions Formula: total width = width + padding + border
Width/height includes content and padding (but not border) Was part of early CSS3 drafts but removed from the spec Only supported in some older browsers (Firefox until version 50)
